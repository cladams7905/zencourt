Image text overlay templates:

- Use only layered templates (non-simple):
  - `sandwich`: headline with optional `accent_top` and optional `accent_bottom`
  - `accent-headline`: headline + one accent line
- Do not use a simple headline-only template for image carousel slides.
- Keep overlay text short and visual:
  - `headline`: 1-4 impactful words
  - `accent_top`: 3-6 words when used
  - `accent_bottom`: 3-8 words when used
- Inline italic emphasis is allowed and encouraged for image overlays:
  - Use `*word*` or `*short phrase*` inline only (never standalone lines).
  - Keep italics sparse (about 1 emphasized fragment per overlay line).
