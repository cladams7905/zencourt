## Listing Subcategory: Property Features

### Objective

Showcase standout property features that motivate showings.

### Required Direction

- Spotlight one hero feature or one tightly related feature set.
- Translate features into buyer value, not only specs.
- Use specific property facts from listing data when available.
- Keep structure scannable, for example short bullets or short paragraphs with line breaks.
- End with one CTA to see the home in person.
